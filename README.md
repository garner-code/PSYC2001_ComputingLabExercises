# Exercise scripts for Data analysis skills for Psychologists

Kelly G. Garner  

License: GGPLV3

This repository contains the scripts containing the exercises outlined in the bookdown project that is a guide to [data analysis skills for psychologists, using R and R Studio.](https://garner-code.github.io/PSYC2001_ComputingLabBook/) 

The book will introduce you to the skills required to analyse data collected using quantitative methods of inquiry in Psychology, including data wrangling, visualisation, and applying inferential tests.  

Each chapter of the book has exercises to work through. Each chapter is associated with a specific project folder, that contains a script for you to work on, as well as the data and other relevant files that you will need to complete the exercises. The project folders can be found below.  

If you find any bugs, or have any suggestions, please add them as an issue on [this github repository.](https://github.com/garner-code/PSYC2001_ComputingLabBook)
